# Client Contact Manager

A pure PHP application for managing clients and contacts with many-to-many relationships.

### By Thembinkosi Ndlovu

## Features

### ✅ Client Management
- Create new clients with auto-generated codes (e.g., FNB001, PRO001)
- View all clients with contact counts
- Link/unlink contacts via AJAX
- Tabs interface for General info and Contacts

### ✅ Contact Management
- Create new contacts with email validation
- Unique email enforcement
- View all contacts with client counts
- Link/unlink clients via AJAX
- Tabs interface for General info and Clients

### ✅ Technical Highlights
- Pure PHP (no frameworks)
- OOP Implementation with MVC pattern
- Well-structured MySQL database with proper indexing
- AJAX requests for seamless linking/unlinking
- Client-side JavaScript validation
- Server-side validation
- Responsive CSS design
- User-friendly notifications

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web server (Apache/Nginx) or MAMP/XAMPP
